REST API: GoogleFonts

  • Description: Metadata for all families served by Google Fonts.
  • Kind of data of JSON respond: object with 2 properties.
  • Data structure:
    • 1st key-value pair: key named "kind" and a value is a string "webfonts#webfontList".
    • 2nd key-value pair: key named "items" and a value is an array of 1513 objects, which contains objects with information about each font family.:
      • "category" (the font category, such as "sans-serif");
      • "family" (the font family name, such as "Abel");
      • "files" (an object containing URLs for the font files);
      • "kind (the kind of object, a webfont object);
      • "lastModified" (the date the font was last modified);
      • "menu" (the URL for the font menu);
      • "subsets" (an array of the font subsets, such as "latin" or "latin-ext");
      • "variants" (an array of the font variants, such as "regular" or "italic");
      • "version" (the version of the font).
